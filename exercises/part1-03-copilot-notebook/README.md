# แบบฝึกหัดที่ 3: Copilot Notebook — วิเคราะห์ข้อมูลอย่างเป็นระบบ

🔑 **ต้องการ M365 Copilot License**

**Copilot Notebook** คือโหมดการทำงานกับ Copilot แบบ "พื้นที่ถาวร" ซึ่งต่างจาก Chat ตรงที่ข้อมูลที่เราป้อนเข้าไปและผลลัพธ์ที่ได้จะถูกเก็บอยู่ใน Notebook เดิมเสมอ ไม่หายไปตามการสนทนา ทำให้เหมาะกับงานที่ต้องทยอยเพิ่มข้อมูลหรือวิเคราะห์หลายรอบ

ในแบบฝึกหัดนี้ เราจะใช้ Copilot Notebook สำหรับงานวิเคราะห์ข้อมูลสินค้าของ CPAll

---

## Feature 1: เปิดและทำความรู้จัก Copilot Notebook

1. เปิด [https://m365copilot.com](https://m365copilot.com) แล้ว Login ด้วย account องค์กร

2. จากเมนูด้านซ้าย ให้มองหาและกดเลือก ปุ่ม  **+ (New Notebook)**

   ![ภาพ Notebook Menu](./images/notebook-menu.png)

3. กรอกชื่อ Notebook ตามที่ต้องการ เช่น "วิเคราะห์ข้อมูลสินค้าขายดี" แล้วกด **Create**

> 💡 **เคล็ดลับ:** เราสามารถเพิ่มไฟล์ใน M365 ของเรา หรือจะอัพโหลดไฟล์จากส่วนนี้ก็ได้
   ![ภาพ Notebook Create](./images/notebook-create.png)

1. ระบบจะแสดงหน้า Notebook ที่ประกอบด้วย 2 ส่วนหลัก:
   - ด้านซ้าย: **พื้นที่ป้อนข้อมูล (Input Area)** — ใส่เนื้อหาหรือข้อมูลที่ต้องการให้ Copilot วิเคราะห์
   - ด้านขวา: **พื้นที่ผลลัพธ์ (Chat Area)** — แสดงผลการทำงานของ Copilot

   ![ภาพ Notebook Overview](./images/notebook-overview.png)

> 💡 **เคล็ดลับ:** ความแตกต่างสำคัญของ Notebook กับ Chat คือ Notebook ไม่มีประวัติ "ห้องแชท" — ทุกอย่างอยู่ในหน้าเดียว และเราสามารถแก้ไขข้อมูลฝั่งซ้ายแล้วรันใหม่ได้เรื่อยๆ

---

## Feature 2: วิเคราะห์ข้อมูลสินค้าด้วย Notebook

1. ในพื้นที่ด้านซ้าย คลิกเปิดเมนูด้านซ้ายสุด > เลือก **New Page**
   ![ภาพ Notebook Add Page Content](./images/notebook-add-page-content.png)

2. ให้คัดลอกข้อมูลสินค้าตัวอย่างด้านล่างนี้วางลงไป:

   ```
   รายการสินค้าขายดีประจำเดือน พฤษภาคม 2026
   
   1. น้ำดื่ม 7-Select 600ml - ขายได้ 4,200 ขวด - รายได้ 42,000 บาท
   2. เรดบูล 150ml - ขายได้ 3,800 กระป๋อง - รายได้ 45,600 บาท
   3. ข้าวกล่องไก่ย่าง - ขายได้ 1,500 กล่อง - รายได้ 67,500 บาท
   4. ซาลาเปาหมูสับ - ขายได้ 3,200 ชิ้น - รายได้ 48,000 บาท
   5. มาม่ารสต้มยำ - ขายได้ 2,900 ซอง - รายได้ 17,400 บาท
   6. Lay's ออริจินัล - ขายได้ 1,800 ถุง - รายได้ 36,000 บาท
   7. กาแฟ 7-Café ลาเต้ - ขายได้ 980 แก้ว - รายได้ 44,100 บาท
   8. โออิชิ ชาเขียว - ขายได้ 2,100 กระป๋อง - รายได้ 42,000 บาท
   ```

3. ในกล่อง Copilot Chat ด้านล่างหรือด้านขวา ให้ใส่คำถามเพื่อให้ Copilot วิเคราะห์:

   ```
   จากข้อมูลนี้ ช่วยวิเคราะห์ว่า:
   1. สินค้าไหนมีรายได้รวมสูงสุด 3 อันดับแรก
   2. หมวดสินค้าไหนทำรายได้ดีที่สุด
   3. ข้อเสนอแนะเพื่อเพิ่มยอดขายสำหรับเดือนถัดไป
   ```

4. กดปุ่ม **Send** และรอดูผลลัพธ์ที่ด้านขวา

---

## Feature 3: ต่อยอดการวิเคราะห์ในหน้าเดิม

จุดเด่นของ Notebook คือเราสามารถเพิ่มข้อมูลหรือถามต่อยอดได้โดยไม่ต้องเริ่มใหม่

1. ในพื้นที่ป้อนข้อมูลด้านซ้าย ให้เพิ่มข้อมูลต่อท้ายที่มีอยู่:

   ```
   เป้าหมายยอดขายรวมเดือนมิถุนายน: 380,000 บาท
   งบประมาณ Promotion: 15,000 บาท
   ```

2. ในกล่อง Prompt ให้ถามต่อยอด:

   ```
   จากข้อมูลทั้งหมดและเป้าหมายที่กำหนด ช่วยเสนอแผน Promotion 3 รูปแบบที่น่าจะช่วยให้ถึงเป้าได้ โดยอยู่ในงบประมาณที่กำหนด
   ```

3. กดปุ่ม **Send** อีกครั้ง และสังเกตว่าผลลัพธ์ใหม่จะปรากฏต่อยอดในพื้นที่ด้านขวา

> 💡 **เคล็ดลับ:** Notebook เหมาะมากกับงานที่ต้องวิเคราะห์ข้อมูลหลายรอบ เช่น การเตรียมรายงาน, การวางแผน promotion, หรือการสรุปผลการดำเนินงานรายเดือน เพราะเราสามารถแก้ข้อมูลฝั่งซ้ายและกด Send ใหม่เพื่อได้ผลวิเคราะห์ที่อัพเดตได้ตลอด

---

## Feature 4: ใช้ Quick create สร้างสไลด์ PowerPoint อย่างรวดเร็ว

หลังจากเราได้ผลวิเคราะห์และแผน Promotion แล้ว เราสามารถต่อยอดจาก Notebook ไปเป็น Presentation ได้ทันที เพื่อใช้สรุปให้หัวหน้าหรือทีม Marketing ดูภาพรวมได้เร็วขึ้น

1. ในหน้า Notebook เดิม ให้มองหาปุ่ม **Quick create** จากเมนูทางด้านซ้าย แล้วเลือกสร้าง **PowerPoint presentation**

   ![ภาพ Notebook Quick Create](./images/notebook-quick-create.png)

> ⚠️ **หมายเหตุ:** ถ้ายังไม่เห็นปุ่มนี้ใน tenant ของคุณ อาจเป็นเพราะ Feature ทยอยเปิดใช้งาน ให้ข้ามไปขั้นตอนถัดไปของคอร์สก่อนได้

2. หากระบบให้ใส่คำสั่งเพิ่ม ให้ใช้ Prompt ตัวอย่างนี้:

   ```
   สร้าง PowerPoint presentation ภาษาไทยจากข้อมูลใน Notebook นี้ โดยมี 5 สไลด์
   - สรุปภาพรวมยอดขายสินค้าเด่น
   - 3 อันดับสินค้าที่ทำรายได้สูงสุด
   - Insight ที่สำคัญจากข้อมูล
   - แผน Promotion 3 รูปแบบสำหรับเดือนมิถุนายน
   - ข้อเสนอแนะสำหรับผู้จัดการสาขา CPAll
   ใช้น้ำเสียงกระชับ เข้าใจง่าย และเหมาะกับการนำเสนอในที่ประชุม
   ```

3. ระบบอาจจะมีการแสดงตัวอย่างโครงสร้างสไลด์ให้ดูเป็นแนวทางก่อนสร้างจริง ให้ตรวจสอบและปรับแก้ Prompt ตามต้องการ แล้วกด **Create** หรือ **Generate** เพื่อให้ระบบสร้างสไลด์ให้
   ![ภาพ Notebook Question Prompt](./images/notebook-question-prompt.png)

4. กดปุ่ม **Create** หรือ **Generate** แล้วรอให้ระบบสร้าง draft ของงานนำเสนอ

5. ตรวจสอบว่า Copilot สรุปเนื้อหาออกมาเป็นหัวข้อสไลด์ได้ครบหรือไม่ โดยสังเกตเรื่องต่อไปนี้:
   - มีสไลด์สรุปภาพรวมก่อนลงรายละเอียด
   - มีการดึงข้อมูลจาก Notebook ไปใช้จริง ไม่ใช่ข้อความกว้างๆ
   - ข้อเสนอแนะและ Promotion เชื่อมโยงกับข้อมูลยอดขายที่เราวิเคราะห์ไว้

> 💡 **เคล็ดลับ:** ถ้าสไลด์ที่สร้างมายังยาวเกินไป ลองสั่งเพิ่มว่า "ให้แต่ละสไลด์มี bullet ไม่เกิน 3-4 ข้อ" หรือ "ให้เหมาะกับการพรีเซนต์ 5 นาที" เพื่อให้ใช้งานจริงได้ง่ายขึ้น

---

## สรุป

ในแบบฝึกหัดนี้ คุณได้เรียนรู้:
- ความแตกต่างระหว่าง **Copilot Notebook** กับ **Copilot Chat**
- การป้อนข้อมูลและให้ Copilot **วิเคราะห์ข้อมูลสินค้า**ของ CPAll
- การ **ต่อยอดการวิเคราะห์** โดยเพิ่มข้อมูลและถามใหม่ในหน้าเดิม
- การใช้ **Quick create** เพื่อเปลี่ยนผลวิเคราะห์ให้ไฟล์งานต่างๆ
