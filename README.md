# Advanced Microsoft 365 Copilot

ยินดีต้อนรับสู่หลักสูตร **Advanced Microsoft 365 Copilot** สำหรับพนักงาน CPAll 🎉

ในหลักสูตรนี้ คุณจะได้ลงมือปฏิบัติจริงกับ Feature ขั้นสูงของ Microsoft 365 Copilot ตั้งแต่การสร้างและแชร์ Prompt ไปจนถึงการสร้าง AI Agent ของตัวเองด้วย Copilot Studio

---

## 📋 เช็คก่อนเริ่ม

- มี Microsoft 365 account ขององค์กร
- เข้าถึง [https://m365copilot.com](https://m365copilot.com) ได้
- สำหรับ **Part 2** — ต้องมีสิทธิ์เข้าใช้ [Copilot Studio](https://copilotstudio.microsoft.com)

> **⚠️ หมายเหตุ:** แบบฝึกหัดบางส่วนต้องการ **M365 Copilot License** แบบเต็ม (ไม่ใช่ Free) จะมีระบุไว้ชัดเจนในแต่ละ Exercise
